Artistic Artifact: Bee Pollination

The Bee Pollination program gets an input from the user about how many flowers the bee should pollinate.

It then moves to touch or "pollinate" the flowers. After the flowers get pollinated, they grow.


Leprechaun Maze

The Leprechaun Maze program allows the user to use the arrow keys to move a leprechaun through an obstacle course filled with traps.

There is a timer that counts and when the leprechaun reaches the gold at the end, a leaderboard is displayed.
