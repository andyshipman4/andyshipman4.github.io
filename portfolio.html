Artistic Artifact: Bee Pollination

The Bee Pollination program gets an input from the user about how many flowers the bee should pollinate.

It then moves to touch or "pollinate" the flowers. After the flowers get pollinated, they grow.


Leprechaun Maze

The Leprechaun Maze program allows the user to use the arrow keys to move a leprechaun through an obstacle course filled with traps.

There is a timer that counts and when the leprechaun reaches the gold at the end, a leaderboard is displayed.


Spiderman Card

The Spiderman Card is an interactive card that we created so send to a Children's Hospital. It includes a game where spiderman is moved around to defeat the villans

When all the villans are defeated the victory screen comes up and victory music plays. An inspiring quote is also written at the bottom.


Minion Brawl

The Minion Brawl game is a two player game where the minions are moved around the screen by the players. The minions can throw balls at each other and when a minion gets hit, they lose health. The goal of the game is to depleat your opponents health before they do.
